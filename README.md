## **My Messanger**
Andoid messanger app that lets people communicate with each other

**Features:**
 1. login with email or gmail 
 2. see online friends
 3.  chat with any friend
 4. change room chat color
 5. upload images to friends
 6. preview images

**Stack:**
 1. Firebase 
 2. Android
